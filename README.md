🔑 Магия Glassmorphism: 4 ключевых свойства

Вот CSS-свойства, которые создают эффект матового стекла:

css
.review-card {
background: rgba(255,255,255,0.15);
backdrop-filter: blur(20px);
border: 1px solid rgba(255,255,255,0.25);
box-shadow: 0 8px 32px 0 rgba(31,38,135,0.37);
}

📖 Подробный разбор

background  rgba(255,255,255,0.15) Полупрозрачная белая подложка (15% непрозрачности) 
backdrop-filter blur(20px) Размывает фон под элементом (эффект матовости) 
border  1px solid rgba(...) Имитирует блик на кромке стекла, обозначает границы 
box-shadow  0 8px 32px ...  Добавляет глубину, карточка «парит» над поверхностью

⚠️ Важно: backdrop-filter работает только в современных браузерах. Для Safari добавьте -webkit-backdrop-filter.

🎁 Дополнительные свойства

css
border-radius: 24px;    / Скруглённые углы /
overflow: hidden;       / Контент не вылезает за границы /
position: relative;     / Для корректной работы слоёв /
z-index: 1;             / Стекло выше фона /

🎨 Как настроить под себя

Прозрачность фона 0.05 – 0.3 Чем меньше, тем прозрачнее стекло 
Сила размытия  blur(5px) – blur(50px)  Чем больше, тем матовее эффект 
Цвет рамки  rgba(0,0,0,0.2)  Тёмная рамка для тёмной темы 
Радиус скругления  10px – 50px  Менее/более округлые углы 

💡 Советы по использованию

1. Контрастный фон обязателен! Стекло видно только на ярком фоне с градиентами или фигурами.
2. Тестируйте в разных браузерах. backdrop-filter может не работать в старых версиях.
3. Не переусердствуйте с размытием. blur(50px) превратит фон в однородное пятно.
4. Добавляйте тень. Без box-shadow карточка потеряет объём.
